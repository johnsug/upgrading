
## GIS Notes

Coordinates
* Latitude: [-90, 90]
* Longitude: [-180, 180]

Types of shape files
* Raster files = 3d (w/ elevation)
* Vector files = 2d (polygons) / shape file format

Three types of shape files (by extension)
* .shp: the main shape file
* .shx: the position index
* .djf: stores the attributes
